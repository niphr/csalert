<p class="cs-section">Features</p>

<div class="cs-cards">
<div class="cs-card"><div class="cs-card-num">01</div><h3>Short-term trend</h3><p>Fit a rolling regression over a configurable window of recent weeks to classify each period as increasing, not increasing, or decreasing, following the Benedetti (2019) analytics strategy.</p></div>
<div class="cs-card"><div class="cs-card-num">02</div><h3>Baseline detection</h3><p>Compare current counts against a historical baseline using a hierarchical log-linear model, with forecast values and prediction intervals returned as new columns alongside the original data.</p></div>
<div class="cs-card"><div class="cs-card-num">03</div><h3>Doubling days</h3><p>Both trend functions append a doubling-days column that estimates how many days at the current rate it would take for the count to double, giving an interpretable speed-of-change measure.</p></div>
</div>
